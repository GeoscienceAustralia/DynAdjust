46// File Info data:
47// Version identifier (10 character string), e.g. "VERSION "
48// Version value (10 character string), e.g. " 0.1"
49// Created identifier (10 character string), e.g. "CREATED ON"
50// Created value (10 character string), e.g. "2012-11-27"
51// Application identifier (10 character string), e.g. "CREATED BY"
52// Application value (10 character string), e.g. " GEOMAP" or " DYNADJUST"
53//
54// For example, the beginning of the binary stream would have:
55// "VERSION 0.1CREATED ON2012-11-27CREATED BY DYNADJUST"
56// OR
57// "VERSION 0.1CREATED ON2013-03-01CREATED BY THE_APP"
58
